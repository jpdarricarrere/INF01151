# INF01151

## Building

At the root of the project, in a unix terminal run:

`make`

### Server only

`make server`

### Client only

`make client`

## Running

### Server

`make runser`

Or, after building:

`./server`

### Client

`make runcli`

Or, after building:

`./client ADDRESS`

Example:

```
$ make client
$ ./client 127.0.0.1
```

## Cleaning

`make clean`
